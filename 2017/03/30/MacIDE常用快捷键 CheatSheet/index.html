Mac IDE 常用快捷键 CheatSheet

发表于 2017-03-30

tags:

  • IDEA
  • 快捷键

有道是 “工欲善其事必先利其器” 作为一名主攻JAVA的程序员，IDEA就是编写JAVA程序的利器，而使用快捷键，更是可以一定程度地提高编码速度。IDEA不同版本之间快捷键会有点小差异，以下是基于 【IntelliJ IDEA 2016.2.1】版本整理的快捷键,亲测有效。

MAC 快捷键符号

⌘（command）、⌥（option）、⇧（shift）、⇪（caps lock）、⌃（control）、↩（return）、⌅（enter）、⎋（esc）

搜索

操作 快捷键
在当前文件类通过结构体搜索 fn ⌘ F12
快速切换至窗口 fn ⌥ F1
全工程搜索，搜索符合关键词的所有文件 double ⇧
搜索类 ⌘ N
搜索文件 ⇧ ⌘ N
打开最近的文件 ⌘E
列出所有使用该字符的地方 ⌥⌘F7
查看实现 ⌥⌘B

工具窗口

操作 快捷键
打开项目窗口 ⌘1
版本控制窗口 ⌘9
Run 窗口 ⌘4
Debug 窗口 ⌘5
Terminal fn⌥F12

文本编辑

操作 快捷键
移动当前代码行 ⇧⌘↑ ⇧⌘↓
复制一行代码 ⌘D
删除一行代码 ⌘⌫
注释/取消注释一行代码 ⌘/
注释代码块 ⌥⌘/
搜索当前打开的文件中的内容 ⌘F
在当前文件中搜索替换 ⌘R
定位到代码哪一行 ⌘G
打开关闭代码块 ⌘+ ⌘-
周围环绕，自动生成注释、for循环语句等语句 ⌥⌘T
在所有用到该符号的地方高亮显示 fn⌘F7

重构相关

操作 快捷键
重命名，变量、类名等 fn⇧F6
提取变量 ⌥⌘V
提取成员变量 ⌥⌘F
提取常量 ⌥⌘C
提取方法 ⌥⌘M
提取参数 ⌥⌘P
复制一个类到某个包下 F5
移动一个类到某个包下 F6
智能提示 ⌥⏎
格式化代码 ⌥⌘L

调试运行

操作 快捷键
运行 fn⇧F9
调试 fn⇧F10
Step over fnF8
Step into F7
Step out fn⇧F8
